Projects

Hawe

Hawe is a customized and stylized to-do productivity app. Users are given incentives to come back through the badges and character creation items. It was created through the use of Node.js, Express, MySQL, Sequelize, Passport.js, etc.

NYT Scraper

This website allows users to scrape articles from the New York Times, and displays them on the page. This is done through the use of the npm packages cheerio and request. It uses MongoDB as its database.

Readventure

[Currently under Rewrite. Link is to the GitHub Repository]

Readventure is website that allows children to gamify their reading list by logging their current page number into the database. More pages being read allows children to 'level up' and collect prizes. The website was also connected to the google books api, and a dictionary api. The previous build was created through firebase. The new build will have Node.js, passport, and MySQL.
